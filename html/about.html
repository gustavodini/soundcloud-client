SoundCloud Client. Build with Electron.

This is NOT an official product. This project does not attempt to reverse engineer the SoundCloud API or attempt to reimplement any part of the SoundCloud Client. Any communication between the user and SoundCloud servers is handled by official SoundCloud Web itself; this is just a native wrapper for SoundCloud Web, like a browser.
